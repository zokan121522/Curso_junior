
## 🧪 Ejercicio 3 – Componentes anidados con props

### 🎯 ENUNCIADO GENERAL
```bash
Refactoriza tu componente TarjetaProducto separándolo en 3 subcomponentes anidados:
	•	ImagenProducto
	•	InfoProducto
	•	BotonCompra

Cada uno recibirá props específicas. Luego usa estos subcomponentes dentro de TarjetaProducto.
```


### 🧭 PISTAS PARA RESOLVERLO
```bash
	1.	Crea 3 nuevos componentes dentro del archivo TarjetaProducto.jsx.
	2.	Cada componente debe ser una función interna al archivo, con sus propias props.
	3.	Reemplaza el contenido JSX de la tarjeta por la composición de estos subcomponentes.
	4.	Asegúrate de que App.jsx siga funcionando igual.
```
### 🧩 PASOS A SEGUIR SIN EL CODIGO
```jsx

// 📄 1. Creamos el componente principal TarjetaProducto
// 👉 Recibe las props: título, descripción, precio e imagen

// 📦 2. Contenedor general de la tarjeta:
//    - Fondo blanco
//    - Esquinas redondeadas
//    - Sombra
//    - Efecto hover con animación

// 🧱 3. Usamos el subcomponente ImagenProducto para mostrar la imagen del producto

// 🧱 4. Usamos el subcomponente InfoProducto para mostrar:
//      - Título grande y negrita
//      - Descripción en gris suave
//      - Precio en verde con estilo destacado

// 🧱 5. Usamos el subcomponente BotonCompra para mostrar el botón "Comprar"

// 🔹 Subcomponente ImagenProducto:
//      • Recibe prop `imagen`
//      • Ocupa todo el ancho con `w-full`
//      • Altura fija `h-48`
//      • `object-cover` para recortar y centrar bien la imagen

// 🔹 Subcomponente InfoProducto:
//      • Recibe props: `titulo`, `descripcion`, `precio`
//      • Título: texto grande, negrita y color oscuro
//      • Descripción: texto gris claro y tamaño medio
//      • Precio: color verde + formateado con `.toFixed(2)`

// 🔹 Subcomponente BotonCompra:
//      • No recibe props
//      • Botón ancho completo (`w-full`)
//      • Color azul (`bg-blue-500`)
//      • Texto blanco
//      • Hover cambia tono
//      • Bordes redondeados y animación con `transition`

```
### 🧩 CÓDIGO COMPLETO
```jsx
// 📄 src/components/TarjetaProducto.jsx
// 📄 1. Componente principal que recibe las props del producto (título, descripción, precio, imagen)
export default function TarjetaProducto({ titulo, descripcion, precio, imagen }) {
  return (
    // 🧱 2. Contenedor de la tarjeta con fondo blanco, esquinas redondeadas y animación hover
    <div className="bg-white shadow-lg rounded-xl overflow-hidden max-w-sm transition hover:scale-105 duration-300">
      
      {/* 📷 3. Subcomponente para mostrar la imagen */}
      <ImagenProducto imagen={imagen} />

      {/* 🧾 4. Subcomponente que muestra el contenido del producto: título, descripción y precio */}
      <InfoProducto 
        titulo={titulo}
        descripcion={descripcion}
        precio={precio}
      />

      {/* 🔘 5. Subcomponente para el botón de compra */}
      <BotonCompra />
    </div>
  );
}

// 🧱 Subcomponente 1 – Imagen del producto
// 👉 Solo necesita la prop `imagen`
function ImagenProducto({ imagen }) {
  return (
    <img
      src={imagen}
      alt="Producto"
      className="w-full h-48 object-cover"  // Estilo: imagen ocupa todo el ancho, altura fija, recorte automático
    />
  );
}

// 🧱 Subcomponente 2 – Info del producto
// 👉 Recibe: título, descripción y precio
function InfoProducto({ titulo, descripcion, precio }) {
  return (
    <div className="p-6">
      <h2 className="text-xl font-bold text-gray-800">{titulo}</h2>         {/* Título destacado */}
      <p className="text-gray-600 mt-2">{descripcion}</p>                  {/* Descripción suave */}
      <p className="text-green-600 font-semibold mt-2">
        {precio.toFixed(2)} €                                             {/* Precio en verde y con 2 decimales */}
      </p>
    </div>
  );
}

// 🧱 Subcomponente 3 – Botón de compra
// 👉 No necesita props, es genérico
function BotonCompra() {
  return (
    <div className="px-6 pb-6">
      <button className="w-full bg-blue-500 text-white py-2 px-4 rounded hover:bg-blue-600 transition">
        Comprar
      </button>
    </div>
  );
}
```
