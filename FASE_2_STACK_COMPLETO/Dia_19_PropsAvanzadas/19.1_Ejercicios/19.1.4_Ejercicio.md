## 🧪 Ejercicio 4 – Props condicionales (variantes visuales)


### 🎯 ENUNCIADO GENERAL
```bash
Modifica tu tarjeta de producto para que reciba una prop llamada destacado.
	•	Si destacado es true, la tarjeta debe tener un borde dorado (border-yellow-500).
	•	Si destacado es false, tendrá un borde gris claro (border-gray-200).

🔁 Esto te entrena en estilos condicionales con Tailwind y props.
```

### 🧭 PISTAS PARA RESOLVERLO
```bash
	1.	En TarjetaProducto, añade una prop destacado.
	2.	Usa una expresión para elegir la clase border según su valor.
	3.	Combina esa clase con las otras usando template strings o concatenación.
	4.	Pásale destacado={true} a alguna tarjeta desde App.jsx.
```

### 🧩 CÓDIGO COMPLETO
```jsx
// 📄 src/components/TarjetaProducto.jsx
// 📄 1. Componente principal TarjetaProducto con una nueva prop: `destacado`
export default function TarjetaProducto({ titulo, descripcion, precio, imagen, destacado }) {
  // 🎨 2. Creamos una variable condicional para el borde:
  //      - Si destacado es true: borde amarillo
  //      - Si no: borde gris claro
  const bordeClase = destacado ? 'border-yellow-500' : 'border-gray-200';

  return (
    // 🧱 3. Contenedor de la tarjeta con:
    //      - fondo blanco, bordes redondeados, sombra
    //      - borde dinámico según la prop `destacado`
    <div className={`bg-white rounded-xl overflow-hidden max-w-sm border-2 ${bordeClase} shadow-md`}>
      
      {/* 🖼️ 4. Imagen del producto */}
      <img 
        src={imagen} 
        alt={titulo} 
        className="w-full h-48 object-cover" 
      />

      {/* 🧾 5. Contenido del producto: título, descripción, precio */}
      <div className="p-6">
        <h2 className="text-xl font-bold text-gray-800">{titulo}</h2>
        <p className="text-gray-600 mt-2">{descripcion}</p>
        <p className="text-green-600 font-semibold mt-2">
          {precio.toFixed(2)} €
        </p>

        {/* 🔘 6. Botón de compra con hover */}
        <button className="w-full bg-blue-500 text-white py-2 px-4 rounded mt-4 hover:bg-blue-600 transition">
          Comprar
        </button>
      </div>
    </div>
  );
}
```
