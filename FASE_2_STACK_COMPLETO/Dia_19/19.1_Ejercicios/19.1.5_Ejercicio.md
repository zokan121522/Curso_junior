## 🧪 Ejercicio 5 – Listado dinámico desde un array

### 🎯 ENUNCIADO GENERAL
```bash
Simula una tienda que renderiza múltiples tarjetas de producto automáticamente a partir de un array de datos.
Cada tarjeta recibe props como titulo, descripcion, precio, imagen.

👉 Este ejercicio es clave para entender cómo mapear datos reales en interfaces.
```

### 🧭 PISTAS PARA RESOLVERLO
```bash
	1.	Crea un array llamado productos con varios objetos (mínimo 3).
	2.	Usa .map() dentro del JSX para renderizar una TarjetaProducto por cada objeto.
	3.	Asegúrate de pasar una key única (index o id).
	4.	Todos los datos deben venir del array, no escritos directamente.
```

### 🧩 CÓDIGO COMPLETO
```jsx
// 📄 src/App.jsx

import TarjetaProducto from './components/TarjetaProducto'

const productos = [
  {
    titulo: "Auriculares inalámbricos",
    descripcion: "Sonido envolvente y batería de 40h.",
    precio: 59.99,
    imagen: "https://source.unsplash.com/400x300/?headphones"
  },
  {
    titulo: "Mochila urbana",
    descripcion: "Impermeable y cómoda para portátiles.",
    precio: 39.99,
    imagen: "https://source.unsplash.com/400x300/?backpack"
  },
  {
    titulo: "Smartwatch deportivo",
    descripcion: "Monitoriza tu salud y actividad.",
    precio: 89.99,
    imagen: "https://source.unsplash.com/400x300/?smartwatch"
  }
]

export default function App() {
  return (
    <div className="min-h-screen bg-gray-100 p-10 grid gap-6 place-items-center sm:grid-cols-2 md:grid-cols-3">
      {productos.map((item, index) => (
        <TarjetaProducto
          key={index}
          titulo={item.titulo}
          descripcion={item.descripcion}
          precio={item.precio}
          imagen={item.imagen}
        />
      ))}
    </div>
  )
}
```
```jsx
// 📄 src/components/TarjetaProducto.jsx (misma que en ejercicios anteriores)

export default function TarjetaProducto({ titulo, descripcion, precio, imagen }) {
  return (
    <div className="bg-white rounded-xl overflow-hidden max-w-sm border-2 border-gray-200 shadow-md">
      <img src={imagen} alt={titulo} className="w-full h-48 object-cover" />
      <div className="p-6">
        <h2 className="text-xl font-bold text-gray-800">{titulo}</h2>
        <p className="text-gray-600 mt-2">{descripcion}</p>
        <p className="text-green-600 font-semibold mt-2">{precio.toFixed(2)} €</p>
        <button className="w-full bg-blue-500 text-white py-2 px-4 rounded mt-4 hover:bg-blue-600 transition">
          Comprar
        </button>
      </div>
    </div>
  );
}
```
