
## 🧪 Ejercicio 2 – Lista de productos con .map()


### 🎯 ENUNCIADO GENERAL
```bash
Crea una lista de productos simulados (con título, descripción, precio e imagen)
y recórrela con .map() para mostrar varias tarjetas de producto usando el componente TarjetaProducto.
```

### 🧭 PISTAS PARA RESOLVERLO
```bash
	•	Declara un array llamado productos dentro de App.jsx con 3–5 objetos (cada uno debe tener título, descripción, precio e imagen).

	•	Asegúrate de pasar las props correctas (titulo, descripcion, etc.).
	•	Agrega una clase grid con Tailwind para que las tarjetas estén en columnas.
```

### 🧩 CÓDIGO COMPLETO

```jsx
// 📄 1. Importamos el componente que vamos a reutilizar
import TarjetaProducto from './components/TarjetaProducto'

// 📄 2. Creamos un array con productos simulados (cada uno tiene título, descripción, precio e imagen)
const productos = [
  {
    titulo: "Sudadera Frontend",
    descripcion: "Calentita y con código en las mangas.",
    precio: 34.99,
    imagen: "https://source.unsplash.com/400x300/?hoodie"
  },
  {
    titulo: "Taza de café JS",
    descripcion: "Ideal para programar sin dormir.",
    precio: 9.99,
    imagen: "https://source.unsplash.com/400x300/?mug"
  },
  {
    titulo: "Sticker Pack Dev",
    descripcion: "Dale vida a tu portátil con estos stickers.",
    precio: 4.99,
    imagen: "https://source.unsplash.com/400x300/?stickers"
  }
];

// 📄 3. Creamos el componente principal de la app
export default function App() {
  return (
    // 📦 4. Contenedor principal: alto mínimo, fondo gris claro y padding
    <div className="min-h-screen bg-gray-100 p-10">

      {/* 🛒 5. Título del catálogo centrado y con estilo grande */}
      <h1 className="text-3xl font-bold text-center mb-8">
        🛒 Catálogo de productos
      </h1>

      {/* 🧱 6. Contenedor tipo grid responsivo:
            - 1 columna en móvil
            - 2 en pantallas pequeñas
            - 3 en pantallas medianas o más
            - con separación entre tarjetas */}
      <div className="grid grid-cols-1 sm:grid-cols-2 md:grid-cols-3 gap-6">

        {/* 🔁 7. Recorremos el array de productos con .map() y mostramos una tarjeta por cada uno */}
        {productos.map((item, index) => (
          <TarjetaProducto
            key={index}                     // ✅ clave única para React
            titulo={item.titulo}           // 🧾 Pasamos props una a una
            descripcion={item.descripcion}
            precio={item.precio}
            imagen={item.imagen}
          />
        ))}

      </div>
    </div>
  )
}
```
