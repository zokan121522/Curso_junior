## ‚úÖ 1Ô∏è‚É£ Guardar un solo perfil

### üéØ OBJETIVO  
```
Crear un diccionario con tus datos y mostrar claves y valores.
```

```python
# Creo diccionario con datos
perfil = {
    "nombre": "Zokan",   # nombre clave:valor
    "edad": 34,          # edad clave:valor
    "ciudad": "Barcelona" # ciudad clave:valor
}

print("Claves:") # Titulo

for clave in perfil.keys():
    print(clave)
# Analog√≠a: cada clave es como el nombre de una casilla

# Muestro valores
print("\nValores:")
for valor in perfil.values():
    print(valor)
# Analog√≠a: cada valor es lo que hay guardado en la casilla
```
### üß† Analog√≠a de la estanter√≠a:
```
	‚Ä¢	El diccionario es una estanter√≠a con cajas.
	‚Ä¢	Cada clave es la etiqueta de la caja.
	‚Ä¢	Cada valor es lo que hay dentro.
	‚Ä¢	El bucle es como ir abriendo cada caja para ver qu√© nombre tiene o qu√© hay dentro.

‚úÖ SIN BUCLE ‚Üí lo ves todo junto, menos claro.

‚úÖ CON BUCLE ‚Üí lo ves desplegado, ordenado y reutilizable.
```

## ‚úÖ 3Ô∏è‚É£ A√±adir perfiles en bucle


```python
usuarios = []  # Caja vac√≠a

    edad = int(input("Edad: "))  # Pido edad
    ciudad = input("Ciudad: ")  # Pido ciudad

    perfil = {  # Creo ficha
       	# üîë Izquierda Clave: el campo fijo ‚Üí ‚ÄúT√≠tulo‚Äù, ‚ÄúAutor‚Äù, ‚ÄúA√±o‚Äù
	    # üì¶ Derecha Valor: lo que rellenas ‚Üí ‚ÄúEl Hobbit‚Äù, ‚ÄúTolkien‚Äù, ‚Äú1937‚Äù

    print(usuario)
# Analog√≠a: cada usuario es una ficha; la lista es la caja
```

## ‚úÖ 5Ô∏è‚É£ Contar cu√°ntos usuarios son mayores de edad

### üéØ OBJETIVO  
```
Contar cu√°ntos perfiles tienen edad >= 18.
```
```python
usuarios = [
    {"nombre": "Zokan", "edad": 34},
    {"nombre": "Ana", "edad": 28},
    {"nombre": "Luis", "edad": 17}
]

contador = 0  # Empiezo conteo

for usuario in usuarios:
    if usuario["edad"] >= 18:  # Si cumple condici√≥n
        contador += 1  # Sumo 1

print("Mayores de edad:", contador)
# Analog√≠a: reviso cada ficha y sumo si cumple el requisito
```
