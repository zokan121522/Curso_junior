## ✨ Parte 1: LISTAS – Tu mochila de Python

Una **lista** es como una **mochila** donde guardas cosas: puede contener números, textos, o incluso otras listas.

```python
mochila = ["cuaderno", "bolígrafo", "bocadillo"]
```

Puedes acceder a lo que hay dentro con su **posición (index)**:

```python
print(mochila[0])  # cuaderno
print(mochila[-1]) # bocadillo (desde el final)
```

---

## 🛠️ Operaciones básicas con listas

```python
# Crear lista vacía
cosas = []

# Añadir elementos
cosas.append("llaves")
cosas.append("teléfono")

# Cambiar un valor
cosas[1] = "móvil"

# Eliminar
cosas.remove("llaves")

# Tamaño de la lista
print(len(cosas))
```

**Analogía:** la lista es una caja, `append()` mete cosas, `remove()` saca cosas, y `len()` cuenta cuántas cosas hay.

---

## 🔄 Parte 2: BUCLE FOR

Un **bucle for** repite una acción para cada elemento de una lista. Es como decir:

> "Por cada cosa en mi mochila, haz esto."

```python
mochila = ["libro", "agua", "sandwich"]

for objeto in mochila:
    print("Tienes:", objeto)
```

**Visual:**

```text
for = bucle = cinta transportadora
cada elemento = una caja sobre la cinta
```

---

## 🌬️ Parte 3: BUCLE WHILE

`while` se repite **mientras** se cumpla una condición. Ideal cuando no sabes cuántas veces se repetirá.

```python
contador = 0

while contador < 5:
    print("Contando:", contador)
    contador += 1
```

**Analogía:** `while` es como un reloj de arena que sigue girando mientras haya arena arriba (condición True).

---

## 🔎 Parte 4: Recorrer listas con for + if

```python
numeros = [4, 7, 2, 8, 5]

for n in numeros:
    if n % 2 == 0:
        print(n, "es par")
    else:
        print(n, "es impar")
```

## ✅ Script 1 – Calculadora de IMC (Índice de Masa Corporal)

### 📘 ¿Qué es el IMC?

Es un valor que estima la **relación entre peso y estatura** de una persona:

\`\`\`
IMC = peso (kg) / estatura² (m²)
\`\`\`

---

### 🧠 Analogía:

Piensa en el IMC como un **semáforo de salud**:

- Verde → Peso saludable
- Amarillo → Riesgo moderado
- Rojo → Riesgo alto

---

### ⚙️ Código paso a paso:

```python
def calcular_imc(peso, estatura):
    imc = peso / (estatura ** 2)
    return round(imc, 2)

def interpretar_imc(imc):
    if imc < 18.5:
        return "Bajo peso ❗"
    elif imc < 24.9:
        return "Peso normal ✅"
    elif imc < 29.9:
        return "Sobrepeso ⚠️"
    else:
        return "Obesidad 🚨"

print("🧮 Calculadora de IMC")

peso = float(input("Introduce tu peso en kg: "))
estatura = float(input("Introduce tu estatura en metros: "))

imc = calcular_imc(peso, estatura)
resultado = interpretar_imc(imc)

print(f"
Tu IMC es {imc}")
print(f"Resultado: {resultado}")
```

## ✅ Script 2 – Calcula tu año de nacimiento

### 🧠 Idea base:

Si conoces tu edad actual, puedes restarla al año actual para saber en qué año naciste.

### 🕰️ Ejemplo:

> Si hoy es 2025 y tienes 30 años → naciste en 1995

---

### ⚙️ Código completo:

```python
from datetime import datetime

def calcular_nacimiento(edad):
    año_actual = datetime.now().year
    return año_actual - edad

print("📅 Calculadora de año de nacimiento")

edad = int(input("¿Cuántos años tienes? "))
año_nacimiento = calcular_nacimiento(edad)

print(f"Naciste en el año {año_nacimiento}")
```
